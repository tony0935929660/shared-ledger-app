# UI Guideline

## Design Direction

- 簡潔
- 商務
- Mobile First
- 高資訊可讀性
- 避免複雜動畫

## Transaction Type Labels

- 收入
- 支出
- 資本投入

## Payment Method Labels

- 公司直接付款
- 成員代墊

## Reimbursement Labels

- 未返還
- 已返還

## Form Order

1. 類型
2. 項目名稱
3. 金額
4. 分類
5. 日期
6. 付款方式（僅支出）
7. 代墊人（條件顯示）
8. 備註
9. 圖片
10. 儲存

## Confirmation

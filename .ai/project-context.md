# Project Context

## Product

Company Ledger 是一套給小型創業團隊使用的共同記帳工具。

## Core Problem

創業初期常見以下問題：

- 多位成員共同支付公司費用。
- 成員可能先替公司代墊。
- 團隊難以確認哪些代墊已返還。
- 收入、支出與資本投入容易混在一起。
- 缺乏建立者、建立時間與刪除紀錄。

## V1 Goal

提供一套低成本、手機與桌機皆可使用、具基本稽核能力的共同記帳工具。

## Primary Users

- 公司創辦人
- 共同創業成員
- 經授權的內部成員

## Non-Goals

V1 不包含：

- 正式會計傳票
- 稅務申報
- 多公司
- 多幣別
- OCR
- AI 分類
- 銀行串接
- 薪資系統
- 股權比例計算

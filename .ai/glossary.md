# Glossary

| 中文 | 英文 | 定義 |
|---|---|---|
| 收入 | Income | 公司營業或其他來源取得的款項 |
| 支出 | Expense | 公司支付或應由公司承擔的費用 |
| 資本投入 | Capital Contribution | 創辦人或成員投入公司的初始或後續資金 |
| 公司直接付款 | Company Payment | 從公司帳戶、公司卡或公司現金直接支付 |
| 成員代墊 | Member Advance | 成員先以個人資金支付公司支出 |
| 代墊人 | Advance Member | 實際先支付該筆公司支出的成員 |
| 返還日期 | Returned At | 公司將代墊款返還給代墊人的日期時間 |
| 軟刪除 | Soft Delete | 保留資料並填入刪除者與刪除時間 |
| 建立者 | Created By | 建立該筆資料的登入使用者 |

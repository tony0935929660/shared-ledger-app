# Git Convention

## Branch

```text
feat/spec-001-foundation
feat/spec-002-transaction
fix/transaction-soft-delete
```

## Commit

```text
feat(auth): add Supabase login flow
feat(transaction): add member advance creation
fix(transaction): prevent non-expense payment method
test(transaction): add reimbursement acceptance tests
docs(spec): update database constraints
```

## Pull Request

每個 PR 原則上只對應一個 Spec 或一個明確修正。
